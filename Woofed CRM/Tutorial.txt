Para iniciar a instalação assista o video do Chatwoot White Label, assim sua VPS vai estar pronta para receber o Woofed CRM

Link Chatwoot: https://www.youtube.com/watch?v=F3iVIHKFmxk&t=1726s&ab_channel=AstraOnline

Acesse o Postgre e Crie o Banco de Dados

psql -U postgres

create database woofedcrm;

Após o deploy migre o banco de dados executando no conteiner principal o comando:

rails db:migrate
